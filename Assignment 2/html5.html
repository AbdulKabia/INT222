HTML5 new form attributes

HTML5 introduced new attributes that can be used along with the form input tag. These new attributes can help make your forms appear and behave any which way you'd like.

How to set up Autofocus


<input type="text" name="entry01" id="entry01" class="htmlForms" autofocus="autofocus" />

How to set up a Placeholder


<input type="text" name="entry02" id="entry02" class="htmlForms" placeholder="This is a placeholder"/>

How to set required for an entry


<input type="text" name="entry03" id="entry03" class="htmlForms" required="required" title="This field is required!"/>
Click this button to see the required field in action

How to create a list/datalist


Code for list on left:
<select name="entry04" class="htmlForms">
<option value="">Select one</option>
<option value="APC100">APC100</option>
<option value="EAC150">EAC250</option>
<option value="ICA002">ICA002</option>
<option value="IPC144">IPC144</option>
<option value="ULI101">ULI101</option>
<option value="IOS110">IOS110</option>
</select>

Code for datalist on right:
<input type="text" class="htmlForms" list="subjectNames" placeholder="Select one" name="entry04.5" />
<datalist id="subjectNames">
<option value="APC100">APC100</option>
<option value="EAC150">EAC250</option>
<option value="ICA002">ICA002</option>
<option value="IPC144">IPC144</option>
<option value="ULI101">ULI101</option>
<option value="IOS110">IOS110</option>
</datalist>

How to ensure a valid email address format


<input type="email" name="entry05" id="entry05" class="htmlForms"/>
Enter some information on the required field above, then enter any email address in this field WITHOUT the "@" symbol and click the submit button to see this field in action.

How to ensure a valid url format


<input type="url" name="entry06" id="entry06" class="htmlForms"/>
Enter some information on the required field above, then enter any random text in this field and click the submit button to see this field in action.
